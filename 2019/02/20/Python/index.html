#Python

##简介
Python 是一个高层次的结合了解释性、编译性、互动性和面向对象的脚本语言。
Python 的设计具有很强的可读性，相比其他语言经常使用英文关键字，其他语言的一些标点符号，它具有比其他语言更有特色语法结构。
• Python 是一种解释型语言： 这意味着开发过程中没有了编译这个环节。类似于PHP和Perl语言。
• Python 是交互式语言： 这意味着，您可以在一个Python提示符，直接互动执行写你的程序。
• Python 是面向对象语言: 这意味着Python支持面向对象的风格或代码封装在对象的编程技术。
• Python 是初学者的语言：Python 对初级程序员而言，是一种伟大的语言，它支持广泛的应用程序开发，从简单的文字处理到 WWW 浏览器再到游戏。

##note
1、删除字符串开头、末尾空白lstrip（）、 rstrip（） 【暂时性的，再次询问，依然存在】若要永久删除，必须将删除操作的结果存回到变量中
2、title（）以首字母大写
3、upper（）全大写
4、lower（）全小写
5、两个乘号表示乘方：32=9 33=27
6、str（）转换为字符串
7、del删除不能使用，pop（）删除也可使用，不知道位置只知道值用remove（），删除值也可接着使用
8、sort（）永久修改列表元素排列顺序，sort（reverse=True）永久修改与字母顺序相反的
9、sorted（）对列表进行临时排序，sorted（reverse=True）临时性修改与字母顺序相反的
10、reverse（）永久性的不按字母顺序，只反转列表元素排列顺序，若想恢复再次调用reverse（）即可
